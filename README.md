# Machine Learning from Scratch: Decision Tree & Random Forest

## Overview
This project implements **Decision Tree and Random Forest** classifiers from scratch in Python to predict mushroom edibility based on structured attributes. The objective is to reproduce core machine learning algorithms and evaluate their performance through an end-to-end modeling pipeline.

---

## Key Features
- Built Decision Tree and Random Forest models from scratch in Python (without using libraries such as scikit-learn)  
- Implemented data preprocessing, feature handling, and stratified train-test split  
- Designed custom split criteria using Gini impurity, entropy, and classification error  
- Developed cross-validation and hyperparameter tuning pipeline  
- Evaluated model performance using standard classification metrics (accuracy, precision, recall, F1-score)  

---

## Objective
To implement fundamental machine learning algorithms from first principles and compare the performance of a single Decision Tree model versus an ensemble Random Forest approach.

---

## Workflow
- Data cleaning and preprocessing  
- Feature and target separation  
- Stratified train-test split  
- Model implementation (Decision Tree & Random Forest)  
- Cross-validation  
- Hyperparameter tuning  
- Model evaluation and comparison  

---

## Results
Both models achieved strong predictive performance, with the Random Forest demonstrating improved generalization compared to a single Decision Tree.

---
